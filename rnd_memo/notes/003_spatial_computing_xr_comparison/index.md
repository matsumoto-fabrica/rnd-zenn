# Android XR vs Apple Vision Pro — 空間コンピューティングの方向性比較

## 仮説

Google Android XR と Apple Vision Pro は、空間コンピューティングに対して根本的に異なるインタラクション設計思想を持っている。Apple は「視線追跡 + ハンドジェスチャー」、Google は「音声ベース」を軸にしており、この違いがアプリ設計・UX・開発者エコシステムに大きな影響を与えるのではないか。
